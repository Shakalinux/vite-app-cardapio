# Vite App Cardápio

Construção de um simples cardápio, utilizando vite como tecnologia.

## Funcionalides 
Algumas funcionalidades como: 
 - Adição do pedido
 - Cálculo Total do Pedido
 - Contagem de Seleção
 - Remoção de itens

Realizando com javaScript, html,css e bootstrap.
